## Project Description
This is a single-page interactive resume for **IT 223 – Web Systems and Technologies** at USTP.  
It features a modern welcome page and multiple interactive elements using JavaScript and jQuery.

## Features
- Welcome prompt that asks for visitor's name
- Beautiful welcome hero page
- Smooth scrolling navigation
- Dark mode toggle
- Animated skill progress bars
- Clickable skill items with animation
- Live project search filter
- Working contact form with validation

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Box Model)
- JavaScript
- jQuery (via CDN)
