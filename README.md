## Project Description
This is a single-page interactive resume built as a requirement for **IT 223 – Web Systems and Technologies** at USTP.  
It looks like a real professional resume but includes multiple interactive features powered by JavaScript and jQuery.

## Features
- Clean and modern design with welcome page
- Smooth scrolling navigation (anchor links)
- Dark mode toggle
- Animated skill progress bars
- Live project search filter
- Clickable skill items with animation
- Working contact form with validation and feedback
- Responsive and professional layout

## Technologies Used
- HTML5
- CSS3 (Flexbox + Grid + Box Model)
- JavaScript (DOM manipulation, events, validation, etc.)
- jQuery (via CDN)
